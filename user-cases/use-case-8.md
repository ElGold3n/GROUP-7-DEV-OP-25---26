USE CASE: 8 View Language Speaker Statistics

CHARACTERISTIC INFORMATION  
Goal in Context  
As a data analyst I want to view the number of speakers for key languages so that I can analyze global linguistic demographics.

Scope  
World Population Reporting System.

Level  
Primary task.

Preconditions  
Language data exists in the database.

Success End Condition  
A report is displayed showing speaker counts and percentages of world population.

Failed End Condition  
No report is produced.

Primary Actor  
Data Analyst.

Trigger  
User selects language report.

MAIN SUCCESS SCENARIO  
System queries speaker counts for key languages.  
System calculates percentage of world population.  
System sorts results by speaker count.  
System displays report.

EXTENSIONS  
Missing language data: system skips and notifies user.

SUB-VARIATIONS  
None.

SCHEDULE  
DUE DATE: Release 1.0
