Welcome to the World Reporting App!

Use the sidebar to choose a report type (Countries, Cities, Capitals, Populations, or Languages). Then:

  1. Select a Scope (Global, Continent, Region, Country, or District).
  2. Pick a Scope Name from the dropdown.
  3. (For Cities) choose a District if needed.
  4. Adjust Top‑N for backend results and Rows per page for how many you see at once.
  5. Click Apply to load your report.
